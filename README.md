# absolute-top.js

get absolute top of element

## Installation

```sh
$ npm install sasaplus1-prototype/absolute-top.js
```

## Usage

via `require()`

```js
var absoluteTop = require('absolute-top');
```

via `<script>`

```html
<script src="absolute-top.min.js"></script>
```

### Example

```js
var div = document.getElementById('div');

absoluteTop.get(div);
```

## Functions

### get(element)

- `element`
  - `Element`

return `top` value.
